Configuración inicial de Firebase (solo se hace una vez)

📋 Paso 1 — Crear proyecto en Firebase

Ve a console.firebase.google.com, inicia sesión con Google, crea un proyecto y sigue los pasos. Plan Spark (Gratuito).

🔐 Paso 2 — Activar autenticación

Authentication → Sign-in method → activa Correo electrónico/Contraseña → Guardar.

🗄️ Paso 3 — Crear base de datos Firestore

Firestore Database → Crear base de datos → Modo de prueba → Habilitar.

⚙️ Paso 4 — Obtener claves

⚙️ Configuración del proyecto → Tus apps → icono </> → Registrar app → copia los valores de firebaseConfig.
